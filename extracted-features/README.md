# Extracted Features

This directory contains code extracted from the PlainApp project for reference.

## Contents

- **screen-mirror/** — Real-time screen streaming feature (H.264 + Opus via MediaProjection)
- **screenshot-detection/** — EXIF-based screenshot metadata detection

Each subdirectory contains:
- Source files in their original package structure
- `HOW-IT-WORKS.md` explaining architecture, data flow, and dependencies
- Excerpt files for integration points (MainActivity, manifest, preferences, etc.)
